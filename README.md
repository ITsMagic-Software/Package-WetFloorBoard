# WetFloorBoard
Hello, so glad you are here! These are some props I made for a capstone class that requires the students to make a game under the game jam timeframe. I modeled these props and textured a few.
I really enjoyed making these props for serval reasons, one being that I have never gone for a realistic style and I feel like I nailed it! the realistic art style was a challenge at first but after some time i became faster and was able to create what you see today.
If you would like to check out the game to play yourself you can go to https://creaturemoon.itch.io/thebrimstone
